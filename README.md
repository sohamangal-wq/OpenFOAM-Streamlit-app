# OpenFOAM-Streamlit-app-created-with-Python-Gemini-collab
Automation of OpenFOAM v13 closed room batch leak simulations and visualization within a web-app made using Streamlit web app as the host
All Python script files created by prompting Gemini pro
All OpenFOAM case files while running a simulation get crated automatically by the Python scripts
The aim is to automate the use of OpenFOAM V13 software as much as possible and enable continuous overnight simulation runs  for various leak scenarios defined inside a closed volume/room that has HVAC inlet outlet vents.
**My project: I am performing toxic and flammable gas leak simulations in a cubical room that has air ventilation inlet and outlet and a wall thickness for the room. I aim to measure the gas concentrations of the gases at the outlet vent using probes and there can be single or multiple leaks with different release properties like temp, pressure, leak orientation, gas species, concentration, molecular weight etc. The HVAC air has some fixed velocity. I also want to do gas mapping for open to air oil & gas or chemical plants (I will import stl of the plant). I'm building a Streamlit UI that uses Python language scripts to help me run my OpenFOAM cases through the Streamlit interface. **
